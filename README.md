# SQL-Queries

This repository contains sample employee datasets in tabular format, ideal for practicing data analysis, SQL queries, data cleaning, and visualization techniques. The datasets include various attributes such as employee ID, name, age, department, city, and salary. These samples are useful for learning how to handle real-world data inconsistencies, missing values, and performing departmental analysis.

📂 Files Included:
1  – Contains employee records from the HR department in Karachi, with ages ranging from 26 to 28 and salaries between 54,000 and 57,500.

2 – A mixed-department dataset including employees from HR, Finance, IT, Marketing, and Sales, with varied ages and salaries.

3 – Focuses on IT department employees, showing a consistent salary trend and department-specific data.

4  – A dataset with some inconsistencies (e.g., missing age marked as *), useful for data cleaning exercises.

🎯 Use Cases:
Practicing SQL SELECT, WHERE, JOIN, and GROUP BY queries.

Performing data wrangling and handling missing values.

Exploring salary distribution across departments and cities.

Creating visualizations like bar charts, histograms, and heatmaps.
